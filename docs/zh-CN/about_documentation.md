# 关于本文档

欢迎阅读 **warpo** 的官方文档！

**warpo** 是一个面向 WebAssembly 的 TypeScript-like 语言编译器与生态（基于 AssemblyScript 分支演进）。

> 说明：当前中文文档仍在持续完善中；若遇到缺失内容，可先参考英文版页面。

## 参与贡献

使用过程中遇到问题，欢迎到 GitHub 提交 issue 或参与讨论：

- https://github.com/wasm-ecosystem/warpo

## 稳定性（Stability）

本文档中的部分章节会标注稳定性等级：

- stability<0>：Deprecated（已弃用）。特性不再维护，未来可能移除。
- stability<1>：Experimental（实验性）。特性处于快速迭代阶段，可能不遵循语义化版本的兼容约束。
- stability<2>：Release Candidate（候选发布）。接口基本稳定，但仍可能发生小概率破坏性变更。
- stability<3>：Stable（稳定）。

## 版本（Versioning）

我们不会维护多个版本的文档；每个章节会注明最低可用版本（默认沿用原 AssemblyScript 的最低版本要求）。

warpo 的首个可用版本为 **2.2.3**。
