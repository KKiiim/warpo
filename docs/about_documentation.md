# About this documentation

Welcome to the official documentation for **warpo**!

**warpo** is a compiler and ecosystem for typescript-like language targeting to WebAssemblyscript bytecode.

## Contributing

If you encounter any problems while using the software, please submit an issue or participate in the discussion on [our github](https://github.com/wasm-ecosystem/warpo).

## Stability

Throughout the documentation are indications of a section's stability.

Some APIs are so proven and so relied upon that they are unlikely to ever change at all.
Others are brand new and experimental,

- stability<0>: Deprecated. The features is not maintained and may remove in the future.
- stability<1>: Experimental. The features is under active development and nearing minimum viability. _It is not subject to semantic versions rules_.
- stability<2>: Release Candidate. The features interface is stable enough and no further breaking changes are anticipated. _It still have small probability of breaking changes occurring_.
- stability<3>: Stable.

## Versioning

Throughout the documentation are indications of a section's versioning.

We will not maintain multiple version's documents, so that each section will contain a minimum usable version (by default is carry over original AssemblyScript).

The first usable version of warpo is **2.2.3**.
